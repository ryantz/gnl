# get_next_line

>[!WARNING]
>segfault always

---

## Project Description
The point of this project is to get the next line from a file specified.

---

## How to install and run the project
Compile the program into an executable:
```bash
cc -Wall -Wextra -Werror get_next_line.c get_next_line_utils.c -o gnl
```
and run
```bash
./gnl
```
